    Constants

    ATTRIBUTE_TYPES

    ATTRIBUTE_TYPES a hash that describes the type of each of the model's fields.

    Each different type represents an Administrate::Field object, which determines how the attribute is displayed on pages throughout the dashboard.

    COLLECTION_ATTRIBUTES

    COLLECTION_ATTRIBUTES an array of attributes that will be displayed on the model's index page.

    By default, it's limited to four items to reduce clutter on index pages. Feel free to add, remove, or rearrange items.

    FORM_ATTRIBUTES

    FORM_ATTRIBUTES an array of attributes that will be displayed on the model's form (`new` and `edit`) pages.

    SHOW_PAGE_ATTRIBUTES

    SHOW_PAGE_ATTRIBUTES an array of attributes that will be displayed on the model's show page.
